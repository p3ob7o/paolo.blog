== Description ==

Night & Day is a WordPress plugin that creates a block allowing users to toggle between two global styles. The block also allows users to select which two global styles to toggle between in the editor.

== Installation ==

1. Upload the `night-and-day` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the Night & Day block in your editor

== Frequently Asked Questions ==

= What are the default styles? =

The default styles are 'night' and 'day'. 'Night' style sets the background color to black and the text color to white. 'Day' style sets the background color to white and the text color to black.

= Can I change the default styles? =

Yes, you can change the default styles in the block settings in the editor.
